Tic Tac Toe – React Project

A simple Tic Tac Toe game built with React as part of a React course by Maximilian Schwarzmüller.

Tech Stack

React

JavaScript (ES6+)

Vite

CSS

Project Overview

This application allows users to:

Edit player names

Play a full Tic Tac Toe game

Switch between active players

Detect and display the winner

Display game logs / move history

The project focuses on core React concepts such as:

State management with useState

Lifting state up

Conditional rendering

Event handling

Component composition

Controlled vs uncontrolled inputs
